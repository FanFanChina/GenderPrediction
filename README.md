# GenderPrediction
A Simple Bayesian Gender Prediction System
## `引言`

根据朴素贝叶斯分类算法而作的性别预测系统

数据来源：网络爬虫


## `实例测试结果`

样本数据：男50人，女50人

男性预测正确率：90.23%

女性预测正确率：81.33%

综合正确率：85.78%
